# CryptoKey Lab

## Тема индивидуального проекта

**«Разработка программного модуля для алгоритмов шифрования с открытым ключом»**

## Автор

**Нигматова Зиеда**

## Описание

**CryptoKey Lab** — интерактивное веб-приложение (программный модуль), демонстрирующее работу алгоритмов шифрования с открытым ключом. Приложение полностью работает в браузере с использованием Web Crypto API без серверной части.

Программный модуль реализует:
- генерацию пар ключей (открытый и закрытый)
- шифрование сообщений открытым ключом
- расшифрование сообщений закрытым ключом
- создание цифровой подписи
- проверку цифровой подписи

## Реализованные алгоритмы

| Алгоритм | Назначение | Длина ключа | Хеш-функция |
|----------|-----------|-------------|-------------|
| RSA-OAEP | Шифрование / Расшифрование | 2048 бит | SHA-256 |
| RSA-PSS  | Цифровая подпись / Проверка | 2048 бит | SHA-256 |

## Функции модуля

- 🔑 **Генерация ключей** — создание пар ключей RSA-OAEP (для шифрования) и RSA-PSS (для подписи)
- 🔒 **Шифрование** — шифрование сообщений открытым ключом RSA-OAEP
- 🔓 **Расшифрование** — расшифрование сообщений закрытым ключом RSA-OAEP
- ✍️ **Цифровая подпись** — создание подписи закрытым ключом RSA-PSS
- ✅ **Проверка подписи** — верификация подписи открытым ключом RSA-PSS
- 📖 **Теория** — теоретические материалы по криптографии с открытым ключом
- 📊 **Визуальные схемы** — наглядные схемы процессов шифрования и подписи

## Запуск

### Локально

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:3000`

### Без npm

Откройте файл `public/index.html` в браузере.

## Деплой на Vercel

Проект готов к деплою на [Vercel](https://vercel.com):

```bash
npx vercel
```

Или подключите GitHub-репозиторий в панели Vercel.
